Introduction

We will create and study web apps and their security issues.

We will primarily use Python and Flask for our server-side code.

We will also briefly study other popular back-end languages and libraries, including node.js and Django.

Your Front-End

Front-ends are necessary for publicly-accessible web apps. We'll do UIs and APIs.

So we will write some HTML, JavaScript, XML, and JSON.

We won't waste time with anything more than a basic UI. That's for another course. That's why I did not say "CSS".

Our Tools

I like to use VS Code for my development environment. I'll use it in the demos. It's very popular in the wild. Check out survey results on stackoverflow (CAREFUL!!!) and other surveys.

You are free to use whatever development environment you wish.

We will use github.

We will also use client-side Python and some command-line tools like wget and curl and other pentesting tools, and we will use Chrome DevTools.

My Background

I studied udergraduate Computer Science and Mathematics in PhD programs at North Carolina State University, one of the 3 large research universities in Research Triangle Park (RTP) and at UC Riverside.
